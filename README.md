# Instruction to run app locally
## Prerequisites:
To run this app properly, you must first build and run the project at the link below by following the instructions in the README.md file:: https://github.com/MinhPhan2608/InterviewBackend

## 1.Navigate to project directory
```bash
cd IntervieFE
```
## 2. Build and start app with docker
```bash
docker compose -f docker-compose.yml up -d
```

## 3. Wait for docker and enjoy the app
### App is now running on port 5173
